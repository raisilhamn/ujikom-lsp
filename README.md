# Uji Kompetensi Junior Web Developer

Tech stack yang digunakan:

-   Laravel
-   Vue.js
-   Tailwind CSS
-   MySQL
-   Inertia.js

## Development Setup

1. Clone repository ini
2. Install dependencies dengan menjalankan `composer install` dan `npm install`
3. Copy file `.env.example` menjadi `.env` dan sesuaikan konfigurasi database
4. Jalankan Migration dan seeder dengan menjalankan `php artisan migrate:fresh --seed`
5. Jalankan aplikasi dengan menjalankan `php artisan serve` dan `npm run dev`
